# Sistema de Gestion de Inventario 

Aplicación de consola desarrollada como parte de mi formación en la Universidad Tecnológica Nacional. El sistema permite la gestión integral de productos, aplicando conceptos avanzados de Programación Orientada a Objetos (POO) y persistencia de datos.


Basado en mi experiencia previa en control de stock y logística en empresas como Perfumerías Pigmento y CNH , diseñé este software para resolver problemas comunes de trazabilidad y reportes de mercadería.


## 🚀 Funcionalidades

* **Altas, Bajas y Modificaciones (CRUD):** Gestión completa de la entidad Producto.


* **Persistencia en JSON:** Los datos se guardan y cargan automáticamente mediante serialización, garantizando la continuidad de la información al cerrar el programa.


* **Reportes Dinámicos:** Generación de informes sobre el estado actual del inventario.


* **Validación de Datos:** Manejo de excepciones para evitar ingresos de datos erróneos y asegurar la integridad del sistema.

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** C# (.NET Core / Framework).

* **Paradigma:** Programación Orientada a Objetos (Clases, Objetos, Encapsulamiento).

* **Formato de Datos:** JSON (System.Text.Json o Newtonsoft.Json).


* **Herramientas:** Visual Studio, Git para control de versiones.

## 📂 Estructura del Proyecto

* **Models/:** Definición de la clase Producto y lógica de negocio.

* **Services/:** Lógica para la manipulación del JSON y persistencia.

* **Program.Cs:** Interfaz de usuario por consola y menú principal.
